# twihaifuck
ツイ廃とSAENSキンの謎言語(brainf\*\*k系言語)

## 使い方
```bash
# brainf**kをtwihaif**kに変換
twfck convert-twfck ./example/twfck/helloworld.twfck

# twihaif**kを実行
twfck twfck ./example/twfck/helloworld.twfck

# brainf**kをsaens-kinに変換
twfck convert-saens ./example/saens/peach.saens

# saens-kinを実行
twfck saens ./example/saens/peach.saens
```

# ビルド方法
Rustのインストールが必要です。

```bash
git clone https://github.com/nennneko5787/twihaifuck.git
cd twihaifuck
cargo build
```
